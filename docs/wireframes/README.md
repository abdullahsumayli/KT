# Wireframes

This directory will contain UI/UX wireframes for the KitchenTech platform.

## Planned Screens

### Mobile App (Flutter)

1. **Home/Landing** - Welcome screen with login/register
2. **Login** - User authentication
3. **Register** - New user registration
4. **Browse Listings** - Grid/list view of available kitchens
5. **Listing Detail** - Full details of a kitchen
6. **My Listings** - Owner's kitchen listings
7. **Create/Edit Listing** - Form to add/update kitchen
8. **Profile** - User account settings
9. **Booking** - Reserve kitchen time slots
10. **Messages** - Chat between renters and owners

### Web Dashboard (Future)

- Analytics for kitchen owners
- Admin panel for platform management
- Booking calendar view
